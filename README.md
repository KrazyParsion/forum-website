#Forum Website#
Similar to Reddit, a user can do the following:
1. sign up
2. log in/log out
3. create posts
4. edit/delete posts made by the same user
5. view database through /database

#Using the Code#
Try running the run.py file first, if it fails, you can activate the flask environment by $source blog/flask_env/bin/activate$ and download the needed packages by $pip install -r requirements.txt$.
